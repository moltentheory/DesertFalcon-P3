Diretivas para compilação:

Instalar Ruby

Instalar a gema Gosu (gem install gosu)

Executar o interpretador ruby da pasta do projeto (ruby lib/main.rb)


Como Jogar:

Utilize as teclas W, A, S, D para movimentar o Falcon

Para aumentar ou diminuir a altura do Falcon utilize as teclas UP e DOWN

Para atirar pressione a barra de espaço
